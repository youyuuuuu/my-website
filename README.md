# 日檢單字模擬測驗系統 (JLPT Test)

本專案採跨領域學習，結合 Python 爬蟲技術與 Flask 網頁框架，解決學習者缺乏即時測驗工具的問題。

## 介紹
- **資料處理**：使用 BeautifulSoup 爬取單字並結構化為 JSON 檔。
- **後端開發**：利用 Flask 串接前後端路由。
- **前端網站**：針對 AI 生成初稿進行手動 CSS 微調。
